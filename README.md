# Mac's Talk — Instituto de Inglés Online 100%

Sitio web oficial de **Mac's Talk Institute**, un instituto de inglés 100% online con clases en vivo y personalizadas para jóvenes y adultos.

## Cursos

- 💬 Conversación & Fluidez
- 🎓 Exámenes Internacionales (IELTS, Cambridge)
- 👨🏼 Jóvenes y Adultos
- ✈️ Viajes & Lifestyle
- 🤝 Entrevistas Laborales
- 📚 Clases Particulares

## Modalidades

- **1:1 Personal** — Clases individuales a tu ritmo
- **Grupos Reducidos** — De 3 a 8 personas
- **Plan Intensivo** — 4 clases semanales de inmersión

## Tecnologías

- HTML5, CSS3, JavaScript (vanilla)
- Formulario con [Formspree](https://formspree.io)
- Calendario con [Calendly](https://calendly.com)
- Hosting: próximamente en [macstalkinstituto.com.ar](https://macstalkinstituto.com.ar)
